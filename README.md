# Servidor Protótipos STEM Criar

Este repositório armazena o código do Servidor de controle dos Protótipos desenvolvidos pelo time Stem Criar.

Para rodar o servidor:

1. Baixe o .zip;

2. Descompacte a pasta `servidor-prototipos-main`;

3. Aperte a **tecla Windows**, busque e **abra o Terminal**

4. No Terminal, execute o comando

```cmd
cd /d %USERPROFILE%\Downloads\servidor-prototipos-main\servidor-prototipos-main && run
```

5. Em poucos segundos, o **Servidor STEM Criar** deverá abrir
